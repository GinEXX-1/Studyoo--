# 真题系统 V1

Studyoo 的真题系统目标不是无差别抓题，而是建立可追踪、可练习、可评阅、可复盘的真题库。

## 2026-07-11 产品升级

- 已完整导入《2026 年全国 1 卷高考数学真题（回忆版）》19 题，并保留原 PDF 与原卷页面。
- 题库页支持拖拽 PDF 导入；有文字层时自动抽取文字，同时始终保留页面图像。
- 支持像播放列表一样建立个人题库：手动选题、按知识点 AI 组卷、按个人薄弱项 AI 组卷。
- 点击题库“开始”后进入独立做题页面，主页不再直接显示练习题。
- 个人页展示累计作答、正确率、平均分、个人题库数量、知识点能力和最近练习。
- 题目解析只保留“从零开始问”的思路提示入口，退出登录移动到个人页。

## 当前已完成

- `exam_papers`：试卷表，保存年份、地区、学科、来源、使用边界。
- `exam_questions`：真题表，保存题干、参考答案、题型、知识点。
- `ai_question_profiles`：AI 理解档案，保存核心思想、易错点、命题意图、前置知识。
- `ingestion_jobs`：导入记录，保存来源和导入数量。
- 真题库 API：
  - `GET /exam/papers`
  - `GET /exam/papers/:paperId/questions`
  - `GET /exam/questions/:questionId`
  - `POST /exam/questions/:questionId/profile`
  - `POST /exam/ingest/manual`
  - `POST /exam/ingest/pdf`
  - `GET /exam/ingestion/jobs`
- 个人题库 API：
  - `GET /collections`
  - `GET /collections/:collectionId`
  - `POST /collections`
  - `POST /collections/ai`
  - `POST /collections/:collectionId/sessions`
- 个人能力 API：`GET /profile/stats`
- 导入的真题会自动同步到练习队列，可从真题库点击“做这道题”。
- 练习页支持“换一题”和评阅后的“继续下一题”。
- 未作答前接口不会下发参考答案；提交作答后才显示参考答案用于订正。

## 当前原则

- 不做无边界全网爬虫。
- 所有题目必须记录来源和使用边界。
- 来源不清晰的题，只能作为本地个人学习材料，不作为公开题库分发。
- AI 不“背题”，而是生成题目理解档案，用于练习、评阅和推荐。

## 手动导入格式

参考：

`backend/samples/exam-import.sample.json`

可以在前端“真题库”中点击“导入试卷”，选择 JSON 文件或直接粘贴 JSON。也可以通过接口请求：

```bash
curl -X POST http://localhost:3000/api/v1/exam/ingest/manual \
  -H "Content-Type: application/json" \
  --cookie "token=<登录后 cookie>" \
  --data @backend/samples/exam-import.sample.json
```

## 后续计划

1. PDF 导入：上传 PDF，先抽取文本，人工确认题目切分后入库。
2. OCR 校对：图片题目必须经过人工校对再进入正式题库。
3. 题目去重：用题干哈希和相似度检测避免重复。
4. 合规来源抓取：只对明确允许使用或用户本地授权的来源做采集。
5. 练习推荐：根据学生作答记录，从真题库里推荐下一题。
