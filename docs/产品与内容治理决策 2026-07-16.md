# 产品与内容治理决策

日期：2026-07-16
状态：内测期约束，拿到首轮真实学生数据后复审

## 产品边界

Studyoo 的主链路是“导入自己的试卷 → 独立作答 → AI 评阅 → 订正 → 遮蔽重做”。共享内容是题库供给能力，不是信息流产品，也不以停留时长和浏览量为目标。

因此，“新发现”在内测期更名为“共享题库”，从一级导航移入题库页。管理员爬虫属于后台供给工具，不构成学生端核心功能。

## 五项治理决定

1. 审核机制：网页爬虫和未来公共来源生成的题目一律先进入待审核区。管理员可编辑、拒绝或通过；未通过内容不得出现在学生端。
2. 版权与下架：只允许采集已加入 `DISCOVERY_ALLOWED_HOSTS` 的公开来源，保留来源链接和审核记录。不得发布无授权的完整商业试卷；收到权利人或来源方请求后应下架对应试卷及练习副本。
3. 质量门槛：题干必须完整、公式可读、选项边界明确。AI 置信度只作排序提示，不代替人工判断。答案缺失可以发布，但必须明确标记为“原网页未附参考答案”。
4. 归属与撤回：学生共享题库仍归原创建者；取消共享后停止向新用户展示。已产生的个人作答历史保留，避免破坏学习记录。网页采集内容归为来源内容，不展示为管理员原创。
5. AI 学习边界：采集内容可以进入检索和题目推荐索引；未经权利确认，不作为模型微调训练材料。学生作答和人工修订数据用于个性化前，也必须按用户授权与隐私规则处理。

## 升级为一级入口的条件

只有同时满足以下条件才重新讨论：首轮至少 3 名真实学生完成核心漏斗；共享题库被证明显著提高“开始做题”而非只增加浏览；审核和下架流程有负责人；来源授权边界可以被解释清楚。
